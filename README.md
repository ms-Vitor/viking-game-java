# 🛡️ Vikings vs Goblins | Java Game

Um jogo de plataforma 2D desenvolvido em Java utilizando o ambiente Greenfoot. O jogador assume o papel de um Viking em uma missão épica para derrotar invasores e sobreviver a desafios intensos.

## 🕹️ O Jogo

* O objetivo principal é explorar os cenários, coletar itens e derrotar todos os inimigos para desbloquear o portal mágico e avançar de fase.

## 🎮 Comandos

* Setas (Esquerda/Direita): Movimentação.

* Seta para Cima: Pular.

* Tecla Space: Atacar.

## 💡 Conceitos de Programação Aplicados

Este projeto foi desenvolvido utilizando os pilares da **Programação Orientada a Objetos (POO)**:

* 🏗️ **Criação e Organização de Classes:** Estruturação do projeto dividindo responsabilidades entre diferentes classes (Viking, Inimigos, Itens, Mundo), garantindo um código mais limpo e organizado.

* 🔒 **Encapsulamento:** Uso de modificadores de acesso para proteger os dados dos objetos, garantindo que atributos (como HP e Pontuação) sejam alterados apenas por métodos específicos.

* ⚙️ **Métodos e Atributos:** Implementação de comportamentos dinâmicos (métodos) e características únicas (atributos) para cada elemento do jogo, permitindo uma simulação realista.

* 🤝 **Interação entre Objetos:** Desenvolvimento de lógica para comunicação entre diferentes instâncias, como o Viking interagindo com itens coletáveis ou sofrendo dano ao colidir com inimigos.

* 🚦 **Controle de Fluxo:** Aplicação de estruturas condicionais e de repetição para gerenciar a lógica do jogo, desde a movimentação dos personagens até as regras de vitória e derrota.

* 📂 **Estruturação de Projeto em Java:** Organização completa de um sistema Java, gerenciando pacotes, recursos externos (imagens e sons) e a hierarquia de classes dentro do ambiente Greenfoot.

## 👺 Bestiário (Inimigos)
Cada inimigo possui atributos únicos, exigindo diferentes estratégias:

* **Goblin Comum:** O soldado padrão.

* **Goblin Mini:** Rápido e furtivo.

* **Goblin Resistente:** O "tanque" do exército, com alta vida.

* **Goblin Boss:** O desafio final com 1000 de HP.

## 💎 Itens e Coletáveis

* 💰 **Moedas:** Aumentam a pontuação final.

* 🧪 **Elixir do Poder:** Aumento temporário de força.

* ❤️ **Coração:** Restaura completamente a saúde do Viking.
